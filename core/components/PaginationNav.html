nametypedefaultdescription

total-itemsnumber1The total number of items.
sizePAGINATION_NAV_SIZE"lg"Specify the size of the PaginationNav.
tooltip-alignmentPAGINATION_TOOLTIP_ALIGNMENT"center"Specify the alignment of the tooltip for the icon-only next/prev buttons. Can be one of: start, center, or end.
tooltip-positionPAGINATION_TOOLTIP_POSITION"bottom"Specify the position of the tooltip for the icon-only next/prev buttons. Can be one of: top, right, bottom, or left.
disable-overflowboolean | undefinedIf true, the '...' pagination overflow will not render page links between the first and last rendered buttons. Set this to true if you are having performance problems with large data sets.
