cds-stack

The Stack component is a useful layout utility in a component-based model. This allows components to not use margin and instead delegate the responsibility of positioning and layout to parent components. In the case of the Stack component, it uses the spacing scale from the Design Language in order to determine how much space there should be between items rendered by the Stack component. It also supports a custom `gap` prop which will allow a user to provide a custom value for the gap of the layout. This component supports both horizontal and vertical orientations.
