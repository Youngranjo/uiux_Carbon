The grid component.

Grid

Attributes

nametypedefaultdescription
alignGridAlignmentType | undefinedSpecify grid alignment. Default is center
condensedbooleanfalseCollapse the gutter to 1px. Useful for fluid layouts. Rows have 1px of margin between them to match gutter.
narrowbooleanfalseContainer hangs 16px into the gutter. Useful for typographic alignment with and without containers.
full-widthbooleanfalseRemove the default max width that the grid has set
with-row-gapbooleanfalseAdd a row gap to the grid that matches the current gutter size. This is useful when you want consistent vertical spacing between rows.
